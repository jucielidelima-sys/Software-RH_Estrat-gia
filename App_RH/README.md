# RH Estratégico Industrial 4.0 - Clean Unique Pages

Correção aplicada para evitar conflito de páginas no Streamlit.

## Importante
Apague a pasta antiga do projeto antes de extrair esta versão.
Se ficarem arquivos antigos dentro da pasta `pages`, o Streamlit continua lendo as páginas antigas e o erro volta.

## Rodar
```bash
pip install -r requirements.txt
streamlit run app.py
```
